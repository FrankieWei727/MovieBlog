# MovieBlog
This is a Django project which developed with django-restful-framework and it also implemented React as the front end service.

Demo : http://mlinked.herokuapp.com/article

## Implement Guide
run requirements   

    pip install -r requirements.txt
    npm install 

migrate data
    
    ./manage.py migrate
    
run project
    
    ./manage.py runserver
    npm bulid
    npm start
